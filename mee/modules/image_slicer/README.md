# 图片切片工具使用说明

## 功能特点

- 现代化图形界面（PyQt5），操作简单直观
- 鼠标拖拽选择切片区域
- 自定义切片名称
- 支持多个切片区域，不同颜色显示
- 批量处理整个文件夹的图片
- 自动按原图片名创建子文件夹
- 切片文件命名格式：原图片名-切片名称
- 实时预览切片区域
- 多线程批量处理，不卡界面

## 安装依赖

```bash
pip install PyQt5 pillow
```

## 运行程序

```bash
python image_slicer_qt5.py
```

## 使用步骤

### 1. 加载示例图片

点击"选择示例图片"按钮，选择一张图片作为示例。这张图片用于定义切片区域。

### 2. 定义切片区域

1. 在"切片名称"输入框中输入切片的名称（如：slice1, top, bottom等）
2. 点击"在图片上框选区域"按钮
3. 在右侧图片上按住鼠标左键拖拽，框选出切片区域
4. 松开鼠标，切片区域会被保存并显示在列表中
5. 重复以上步骤添加更多切片区域

提示：
- 切片名称会自动递增（slice1 → slice2 → slice3...）
- 可以随时修改切片名称
- 不同切片区域会用不同颜色显示

### 3. 管理切片区域

- 删除切片：在列表中选中要删除的切片，点击"删除选中的切片"
- 清空所有：点击"清空所有切片"按钮清空所有已定义的切片

### 4. 批量处理

1. 点击"选择输入文件夹"，选择包含待处理图片的文件夹
2. 点击"选择输出文件夹"，选择切片结果的保存位置
3. 点击"开始批量处理"按钮

处理过程中会显示进度条和处理日志。

## 输出结果

处理完成后，输出文件夹的结构如下：

```
输出文件夹/
├── 图片1/
│   ├── 图片1-slice1.jpg
│   ├── 图片1-slice2.jpg
│   └── 图片1-slice3.jpg
├── 图片2/
│   ├── 图片2-slice1.jpg
│   ├── 图片2-slice2.jpg
│   └── 图片2-slice3.jpg
└── ...
```

每个原始图片会创建一个以其文件名命名的文件夹，该文件夹内包含所有切片，切片文件名格式为"原图片名-切片名称.扩展名"。

## 支持的图片格式

- PNG (.png)
- JPEG (.jpg, .jpeg)
- BMP (.bmp)
- TIFF (.tiff, .tif)

## 注意事项

1. 示例图片只用于定义切片区域，不会被切片
2. 批量处理时，程序会根据定义的切片坐标处理输入文件夹中的所有图片
3. 确保输入文件夹中的图片尺寸和示例图片相同或相似，否则切片位置可能不准确
4. 如果输出文件夹中已存在同名文件，会被覆盖
